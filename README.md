# おにいさんずらぶ ファンサイト 💕

「おにいさんずらぶ」を応援する、ファンのためのおうちサイトです。
ブラウザだけで動く**単一HTML**・オフライン可（PWA）・**編集して使えるテンプレート**。

公開URL（GitHub Pages）: https://kajiyaan.github.io/oniisanzu-love-fansite/

## セクション
- 🌟 ヒーロー（タイトル・キャッチコピー）
- 🧑 メンバー紹介（カード型）
- 📷 ギャラリー
- 📰 ニュース＆ヒストリー（年表）
- 💌 ファンの声（メッセージ掲示板）

## 編集のしかた
`index.html` を開いて、次のプレースホルダを差し替えるだけです。
- 「〇〇」「ここに紹介文」→ 実際のテキストへ
- 絵文字（🧑📷）→ 写真は `<img src="pics/xxx.jpg" alt="">` に置き換え（画像は `pics/` に入れる）
- `YYYY.MM.DD` → 実際の日付へ
- 色を変えたいときは、ファイル冒頭の `:root{ --pink: ... }` などの変数をいじるだけ

## アプリとしてインストール（PWA）
- Android(Chrome)：ページを開く → メニュー →「アプリをインストール／ホーム画面に追加」
- iPhone(Safari)：共有 →「ホーム画面に追加」

## データについて
- 「ファンの声」は**サーバー送信なし**。その端末のブラウザ（localStorage）内だけに保存されます。

## GitHub Pages で公開する
1. リポジトリの **Settings → Pages** を開く
2. **Source** を「Deploy from a branch」、**Branch** を `main` / `/ (root)` に設定して Save
3. 1〜2分で上記の公開URLで見られるようになります

## 注意
- **非公式ファンサイト**です。実在の人物・写真・情報を掲載する場合は、必ず許可を取ってください。
